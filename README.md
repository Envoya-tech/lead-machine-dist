# Lead Machine by Envoya

> **We send the agent. You close the deal.**

Lead Machine is an AI-powered outbound sales engine. It automates lead sourcing, personalised email generation, and pipeline tracking — so your team can focus on closing.

## Installation

### macOS
See [INSTALL_MAC.md](./INSTALL_MAC.md) for full step-by-step instructions.

1. Download `lead-machine-x.x.x.pkg`
2. Double-click to open and follow the installer
3. Setup wizard opens automatically in your browser
4. **Estimated time:** 15–20 minutes on a fresh machine

### Linux (Ubuntu 22.04+)
```bash
tar -xzf lead-machine-x.x.x.tar.gz
cd lead-machine-x.x.x
bash installer/install.sh
```

### Windows (run as Administrator)
1. Extract `lead-machine-x.x.x.tar.gz`
2. Right-click `installer\install.bat` → Run as administrator

## What's included

- **AI Email Engine** — personalised cold outreach based on live web signals
- **Lead Sourcing** — Apollo.io integration with ICP scoring
- **Pipeline Tracking** — meetings, follow-ups, reply detection
- **Manager Dashboard** — real-time KPIs for your sales team
- **Self-learning ICP** — automatically improves targeting over time
- **Campaign Management** — run multiple outbound campaigns in parallel

## Requirements

You'll need the following API keys during setup:

- **Anthropic API key** — for AI email generation ([get one](https://console.anthropic.com/settings/keys))
- **Apollo.io API key** — for lead sourcing ([get one](https://developer.apollo.io))
- **Brave Search API key** — for contact research ([get one](https://brave.com/search/api/))
- **Lead Machine license key** — provided by Envoya
